# template-test

> my laudable SAO template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao test my-project
```

### From git

```bash
sao yuridh/template-test my-project
```
